Report № 001  ·  Automation  ·  LLM Pipeline

AI & Data Science Daily Digest

A serverless journalist that reads the AI internet so you don't have to

Production

An automated Google Apps Script pipeline that curates, summarises, and delivers a twice-daily digest of the most important AI & Data Science news directly to your inbox — fully serverless, zero API keys required at startup, running at roughly $0.21 per month in perpetuity.

What It Does

  • Monitors 29 RSS feeds across 5 quality tiers — lab blogs, research, news, DS sites, community
  • Scores every article with a 3-tier weighted keyword system (W3/W2/W1) — zero API cost at this stage
  • Sends top 40 to Gemini 2.5 Flash-Lite for smart deduplication → 15 best articles selected
  • Deep journalist-quality summarisation via Gemini 2.5 Flash identifying up to 5 major breakthroughs
  • Delivers styled HTML email at 6 AM (Morning Edition) and 4 PM (Evening Edition) daily
  • Index-based URL injection — model never hallucinates URLs; pure JavaScript lookup from original array

Engineering Decisions

  • Two-stage pipeline: Flash-Lite for classification (cheap), Flash for generation (quality) — 10× cost savings
  • responseSchema enforcement via Vertex AI — malformed JSON is structurally impossible
  • Date-window filtering via Script Properties — no cross-edition duplicates
  • Per-feed try/catch — one broken feed never blocks 28 others
  • 429 retry with exponential backoff (15s, 30s) built in
  • XML sanitisation handles BOM, control chars, HTML error page detection across all feeds
Pipeline Architecture
29 RSS Feeds (5 Tiers) → fetchRSSArticles() → scoreArticle() W3/W2/W1 → Top 40 → Flash-Lite Filter → 15 → Flash Summarise → URL Injection (JS) → GmailApp.sendEmail()

Report № 004  ·  Agentic AI  ·  Human-in-the-Loop

LangGraph Fact-Checker with HITL

Verification pipelines that pause for human wisdom at exactly the right moment

Agentic

A LangGraph-powered fact-checking pipeline that orchestrates retrieval, reasoning, and verification agents — with strategically placed human-in-the-loop interrupts. The system demonstrates that the most production-ready agentic AI isn't fully autonomous: it knows when to pause, hand off to a human, and resume with enriched context.

Pipeline Stages

  • Claim Extractor: Parses input documents into atomic, verifiable claims
  • Retrieval Agent: FAISS-backed RAG retrieves relevant evidence for each claim
  • Verdict Agent: Reasoning LLM assigns Supported / Refuted / Unverifiable with confidence score
  • HITL Interrupt: Low-confidence or contested claims routed to human reviewer
  • Synthesis Agent: Compiles verified claims into structured fact-check report

LangGraph Implementation

  • State graph with conditional branching — confidence threshold triggers HITL node
  • Persistent graph state enables mid-pipeline human override without restarting
  • LangFuse integration for full observability — every node traced, latency and token counts logged
  • Retry edges handle retrieval failures and LLM timeouts gracefully
  • Modular node design — swap retrieval backend or verdict model independently
